
## Complete guide to building RestApi with typescrit

A comprehensive guide to building RESTful APIs using TypeScript, providing detailed explanations, code samples, and best practices to empower developers in creating robust and scalable backend systems.
## Documentation

1) Initialise the repository: npm init.
2) Install typescript: npm instal -D typescript.
3) Install node for typescript: npm instal -D ts-node.
4) For running project and watching for change install nodemon: npm install -D nodemon
5) Configure typescript by creating tsconfig.json. Specify things you need.
6) Configure nodemon by creating nodemon.json.
